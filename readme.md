##  Features
- Generate phone numbers from patterns like `+21261614xxxx`
- Supports multiple patterns separated by commas
- Generate any number of results
- Copy all generated numbers with one click
- Download results as a `.txt` file
- Clean and modern user interface

##  How to Use
1. Enter how many numbers you want to generate  
2. Enter one or more patterns (comma separated)  
   - Example: `+21261614xxxx,+21261615xxxx`
3. Click **Generate**
4. Copy or download the generated numbers
